The Amazing Dice Game

The rules of the game are:

1. You have to get 5000 points, exactly
2. 1's count as 100, 5's count as 50
3. Three of a kind counts as the number x 100 except 1's which would be 1000
4. Six 1's is an automatic, immediate win
